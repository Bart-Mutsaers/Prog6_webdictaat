WPF Basics

XAML

Bij het opstarten van een WPF project krijg je gratis 2 XAML files.
De vereiste App.xaml en een MainWindow.xaml. Beiden XAML files bevatten een code-behind .cs file.

De aangeleverde MainWindow.xaml file bevat al standaard een titel, breedte, hoogte en Grid als child element. Nu kunnen we zelf een elementen boom gaan opbouwen in deze grid om een volledige user interface te krijgen. Deze boom bestaat uit XML elementen.

Elk XAML element bevat een referentie naar een .NET class. Je kunt dus ook zelf een class omzetten naar een XAML element. De meest voorkomende manier waarop dit gebeurd zijn user controls Het .NET framework bevat al een grote lijst aan voorgedefineerde classes.

Om onze user interface te implementeren kunnen we een onderscheid maken tussen 2 soorten elementen. Hieronder volgen een aantal voorbeelden van beiden type elementen.

Containers

Deze elementen hebben de eigenschap dat ze bedoeld zijn om andere elementen te positioneren. Ze hebben over het algemeen geen 'styling'. Daarnaast kunnen andere elementen doormiddel van 'attached properties' hun positie afhankelijk maken van deze containers.

  • DockPanel

    <DockPanel Width="100" Height="50">
    <Button Content="A" DockPanel.Dock="Left" Width="50"/>
    <Button Content="B" DockPanel.Dock="Right" Width="50"/>
</DockPanel>

  • WrapPanels

    <WrapPanel Width="100" Height="60">
   <Button Content="Left" Width="50"/>
   <Button Content="Right" Width="50"/>
   <Button Content="Left" Width="30"/>
   <Button Content="Right" Width="70"/>
   <Button Content="Right" Width="100"/>
</WrapPanel>

  • StackPanel

    <StackPanel Width="100" Height="120">
   <Button Content="Left" Width="50"/>
   <Button Content="Right" Width="50"/>
   <Button Content="Left" Width="30"/>
   <Button Content="Right" Width="70"/>
   <Button Content="Right" Width="100"/>
</StackPanel>

  • Grid

    <Grid Width="100" Height="50">
   <Grid.RowDefinitions>
       <RowDefinition />
   </Grid.RowDefinitions>
   <Grid.ColumnDefinitions>
       <ColumnDefinition />
       <ColumnDefinition />
   </Grid.ColumnDefinitions>
   
   <Button Grid.Row="0" Grid.Column="0" Content="A" />
   <Button Grid.Row="0" Grid.Column="1" Content="B" />
</Grid>

  • Border

    <Border BorderThickness="5" BorderBrush="Green"
        Width="50" Height="50">
    <Button Grid.Row="0" Grid.Column="0" Content="A" />
</Border>

User Controls

Deze elementen hebben een functie binnen je pagina. Of ze tonen informatie, of geven de gebruiker de mogelijkheid input te leveren.

De C# code die je ziet is gelinked aan een event vanuit de XAML.

  • Button

    <Button Content="A" Click="ClickA"></Button>
    private void ClickA(object sender, RoutedEventArgs e)
{
    //Add Behavior here
}

  • Button

    <ListBox Name="Listy">
    <ListBoxItem>Item 1</ListBoxItem>
    <ListBoxItem>Item 2</ListBoxItem>
</ListBox>
    Listy.Items.Add(new ListBoxItem
    {
        Content = "Item 3"
    });

  • TextBox

    <Grid Width="100" Height="50">
    <TextBox Name="Boxy" Text="Hell world" 
    TextChanged="TextBox_TextChanged"></TextBox>
</Grid>
    private void TextBox_TextChanged(object sender, TextChangedEventArgs e)
{
    Console.WriteLine(Boxy.Text);
};
